# Rishikesh Dhakal - Personal Website

A simple and modern personal portfolio website built with HTML, CSS, and JavaScript.

## 🚀 How to Host on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [GitHub](https://github.com) and sign in
2. Click the "+" icon in the top right and select "New repository"
3. Name your repository: `yourusername.github.io` (replace `yourusername` with your actual GitHub username)
   - Example: If your username is `rishikeshdhakal`, name it `rishikeshdhakal.github.io`
4. Make sure it's set to **Public**
5. Click "Create repository"

### Step 2: Upload Your Files
1. On your repository page, click "uploading an existing file"
2. Drag and drop all these files:
   - `index.html`
   - `style.css`
   - `script.js`
   - `README.md`
3. Scroll down and click "Commit changes"

### Step 3: Enable GitHub Pages
1. Go to your repository settings (click "Settings" tab)
2. Scroll down to "Pages" in the left sidebar
3. Under "Source", select "main" branch
4. Click "Save"

### Step 4: Access Your Website
Your website will be live at: `https://yourusername.github.io`

It may take a few minutes for GitHub to build and deploy your site.

## ✏️ Customization Guide

### Update Your Information

#### In `index.html`:
- Replace "your.email@example.com" with your actual email
- Replace "yourusername" in GitHub and LinkedIn URLs with your usernames
- Update the project descriptions and links
- Modify the About Me section text
- Update skills in the Skills section

#### In `style.css`:
- Change colors by modifying the CSS variables at the top:
  ```css
  :root {
      --primary-color: #2563eb;  /* Change this to your preferred color */
      --secondary-color: #1e40af;
      /* ... other colors */
  }
  ```

### Add Your Projects
1. Replace the placeholder project cards in the Projects section
2. Add real project images (recommended size: 600x400px)
3. Update project titles, descriptions, and links

### Add a Profile Photo (Optional)
1. Add your photo to the repository
2. In `index.html`, add an `<img>` tag in the About section:
   ```html
   <img src="your-photo.jpg" alt="Rishikesh Dhakal" class="profile-photo">
   ```
3. Add styling in `style.css`
